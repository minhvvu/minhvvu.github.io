Description

This is a small part of my current work of analyzing the big data of a telecomunication company in Viet Nam. The purpose of this visualization is to see the change of user data in a period (from June to September 2016).

The data are collected from various sources (from Oracle database, some log-servers farm, other audit systems,... ) and stored on our Hadoop cluster and analyzed with Spark.

With a small subset data (like in this demo), I used some python libraries: pandas for reading data, pyplot for plotting, scikit-learn for some ML tasks. Here I show just the histogram of some common fields (for example: the total amount of charge, the total calling time - in minute, the total number of SMS, ...). The detail info is not explained here. The field name is written in Vietnamese, and the purpose is just a demo!
